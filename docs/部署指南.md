# VideoSum 部署指南

> 打包、分发、安装说明。

---

## 1. 开发环境搭建

### 1.1 后端环境

```bash
# 安装Python 3.10+
# 创建虚拟环境
python -m venv venv
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac

# 安装依赖
cd backend
pip install -r requirements.txt
```

### 1.2 前端环境

```bash
# 安装Node.js 18+
# 安装依赖
cd frontend
npm install
```

### 1.3 开发模式运行

```bash
# 终端1：启动后端
cd backend
uvicorn main:app --reload --port 8000

# 终端2：启动前端
cd frontend
npm run dev
```

---

## 2. 构建打包

### 2.1 构建前端

```bash
cd frontend
npm run build
```

构建产物将输出到 `frontend/dist/` 目录。

### 2.2 配置Electron

```javascript
// electron/main.js
const { app, BrowserWindow } = require('electron')
const path = require('path')

function createWindow() {
  const win = new BrowserWindow({
    width: 1200,
    height: 800,
    webPreferences: {
      nodeIntegration: true
    }
  })
  
  win.loadFile(path.join(__dirname, '../frontend/dist/index.html'))
}

app.whenReady().then(createWindow)
```

### 2.3 打包Electron

```bash
# 安装electron-builder
npm install electron-builder --save-dev

# 打包
npm run build
```

## 5. 后端打包（可选）

```bash
pip install pyinstaller

# 打包后端
cd backend
pyinstaller --onefile --name videosum-api main.py
```

## 6. 安装说明

### 6.1 Windows

1. 下载 `VideoSum Setup 0.1.0.exe`
2. 双击运行安装程序
3. 选择安装目录
4. 完成安装

### 6.2 macOS

1. 下载 `VideoSum 0.1.0.dmg`
2. 双击打开DMG文件
3. 将VideoSum拖入Applications文件夹
4. 首次运行需要在"系统偏好设置 > 安全性与隐私"中允许

### 6.3 Linux

1. 下载 `VideoSum 0.1.0.AppImage`
2. 添加执行权限：`chmod +x VideoSum*.AppImage`
3. 双击运行

## 7. 依赖说明

### 7.1 运行时依赖

| 依赖 | 说明 | 是否必需 |
|------|------|----------|
| FFmpeg | 音视频处理 | 是 |
| yt-dlp | 视频下载 | 是 |
| Ollama | 本地LLM | 可选 |

### 7.2 安装FFmpeg

**Windows**:
```bash
winget install ffmpeg
```

**macOS**:
```bash
brew install ffmpeg
```

**Linux**:
```bash
sudo apt install ffmpeg
```

## 8. 常见问题

### Q: 打包后应用无法启动

**解决方案**：
- 检查FFmpeg是否已安装
- 检查防火墙设置
- 查看日志文件

### Q: 视频下载失败

**解决方案**：
- 检查网络连接
- 尝试使用cookies导入
- 更新yt-dlp版本

### Q: LLM调用失败

**解决方案**：
- 检查API密钥是否正确
- 检查网络连接
- 在配置页面测试连接
