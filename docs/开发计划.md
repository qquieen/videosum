# VideoSum 开发计划

> 项目整体规划、里程碑和时间线。

---

## 1. 项目概述

**项目名称**：VideoSum  
**项目目标**：本地智能视频总结与课件生成工具  
**技术架构**：Electron + Vue 3（前端） + FastAPI + Python（后端）

---

| 任务管理API | ⏳ | `backend/api/task.py` |
| 日志API | ⏳ | `backend/api/log.py` |
| 插件管理API | ⏳ | `backend/api/plugin.py` |
| API测试 | ⏳ | Swagger/Postman |

**复用代码**：
- `models.py` - 数据模型
- `config_manager.py` - 配置管理
- `downloader.py` - 下载模块
- `scheduler.py` - 核心调度
- `asr/` - ASR引擎
- `llm/` - LLM引擎
- `plugins/` - 插件系统
- `extensions/` - 扩展模块
